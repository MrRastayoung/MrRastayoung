Como crear network definida

inspeccionar networks.

·docker network ls

·docker network inspect <id/namenetwork>

Crear nueva network.

·docker network create <newnetwork>

pero nosotros queremos una network personalizada

Crear network personalizada.

·docker network create

--driver= bridge

--subnet=172.22.0.0/16

-o com.docker.network.bridge.enable_icc= true

-o com.docker.network.bridge.enable_ip_masquerade= true

-o com.docker.network.bridge.host_binding_ipv4= 0.0.0.0 <name_network>

Arrancar un nuevo Docker con la nueva Network.

·docker run --net= <name_network> --ip= 172.22.0.12 --name= base -itd altairisx/base:fedora /bin/bash
