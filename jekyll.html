Jekyll y Hide?

Primeros pasos

Conocimientos basicos

  1. Nuevo Espacio web > jekyll new new_blog
  1. Activar server jekyll > jekyll serve >> http://localhost:4000
  1. Estructura del espacio web

-- _config.yml

-- _layouts

-- _posts

-- _site

-- css

-- index.html

config.yml ---> COnfiguración host, puerto, nombre blog

_layouts ---> Diseños personalizados

_posts ---> Articulos escritos en MarkDown o Textile, se compilan y se almacenan en _SITE

_site ---> El navegador web consulta este directorio en busca de las webs compiladas

css y index.html ---> Index y css

  1. Nuevo post > Estructura >> year-month-date-{post-slug}.{file-extension} >> 2014-03-11-hello-world.md > Crea el post y Guardar en _posts >> Metadatos titulo y disposición del contenido >> --- >>layout:post >>title: hello world >> ---
